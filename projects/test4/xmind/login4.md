# 登录注册流程

## 登录

### 手机验证码登录（默认）
- 页面一
	- 唯品会logo
	- 填写手机号（校验手机号的正确格式，输入正确登录按钮即可高亮）
	- 勾选协议（默认不勾选，点击登录按钮判断是否勾选）
	- 第三方登录：微信、QQ、apple
	- 账号密码登录、新用户注册入口
	- 无法登录（在页面右上角，点击唤起底部浮层）
- 页面二
	- 输入验证码（校验6位数字验证码）

### 运营商手机号码一键登录
- 唯品会logo
- 显示本机号码（支持编辑跳转到填写手机号验证码登录）
- 勾选协议（默认不勾选，点击登录按钮判断是否勾选）
- 换个方式登录（点击返回默认登录）

### 账号密码登录
- 填写用户名/绑定手机/绑定邮箱
- 填写密码（密码支持显示或脱敏）
- 勾选协议（默认不勾选，点击登录按钮判断是否勾选）

### 第三方账号登录
- 微信
- QQ
- apple Id

### 注意：各种登录方式之间可以来回切换，但不要使用tab来切换，而是通过换个登录方式跳转页面实现
