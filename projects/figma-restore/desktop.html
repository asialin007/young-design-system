设计理念

年轻版设计系统 2.0 采用极简主义设计理念，强调功能性和视觉清晰度。

核心原则

  • 扁平化设计（无阴影）
  • 直角边框（border-radius: 0）
  • 按钮默认黑色
  • CSS 变量驱动
  • 12 列栅格系统

技术实现

本页面使用年轻版设计系统 2.0 提供的 CSS 变量和组件类构建，确保了代码的可维护性和视觉一致性。 所有颜色、间距、字体均通过 CSS 自定义属性定义，便于主题切换和全局调整。 响应式设计基于 12 列栅格系统，支持移动、平板、桌面三种断点。
