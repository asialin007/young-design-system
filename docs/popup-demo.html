Popup 弹出层组件演示 直角设计

1. 右侧关闭按钮（默认配置）推荐

从底部向上滑出的弹出层，使用右侧关闭按钮。这是最常用的配置，适合一级页面、独立弹窗等场景。

导航栏高度：54px（只有主标题）

2. 左侧返回 + 右侧关闭（二级页面）推荐

从底部向上滑出的弹出层，同时使用左侧返回按钮和右侧关闭按钮。适合二级页面、详情页、筛选页等场景。

导航栏高度：58px（有主副标题）

3. 顶部弹出 - 右侧关闭按钮

从顶部向下滑出的弹出层，使用右侧关闭按钮。适合通知、快捷操作等场景。

导航栏高度：54px（只有主标题）

4. 右侧弹出 - 左侧返回 + 右侧关闭（侧边栏）

从右侧滑出的弹出层，同时使用左侧返回和右侧关闭按钮。常用于设置面板、详情展示等场景。

导航栏高度：54px（只有主标题）

5. 左侧弹出 - 右侧关闭按钮（侧边菜单）

从左侧滑出的弹出层，使用右侧关闭按钮。常用于菜单导航、筛选面板等场景。

导航栏高度：54px（只有主标题）

6. 带底部操作栏的弹出层（右侧关闭）

包含导航栏、内容区域和底部操作栏的完整弹出层，使用右侧关闭按钮。 底部操作栏包含分隔线（视觉装饰）和主次按钮组。

导航栏高度：54px（只有主标题）
底部操作栏：分隔线（135px × 5px）+ 按钮组（次按钮 + 主按钮）
按钮容器内边距：12px 8px 8px（左右 8px，按钮根据屏幕宽度自动等宽分配）

7. 表单示例（左侧返回 + 右侧关闭）

在弹出层中展示表单的完整示例，同时使用左侧返回和右侧关闭按钮，适合二级页面场景。

导航栏高度：54px（只有主标题）

导航栏按钮使用说明

核心规则：
1. ✅ 默认使用右侧关闭按钮（独立使用，最常用）
2. ✅ 左侧返回必须和右侧关闭同时存在（二级页面场景）
3. ❌ 禁止单独使用左侧返回按钮（必须搭配右侧关闭）

导航栏高度规则：
- 只有主标题：54px
- 有主副标题：58px（自动增高 4px）

标题宽度自适应：
- 无按钮：280px
- 仅右侧关闭：200px（默认）
- 左侧返回 + 右侧关闭：160px

❌ 错误示例：单独使用左侧返回按钮

为什么这是错误的？
- 违反了年轻版设计系统的规范
- 用户习惯右侧有关闭按钮（X）
- 左侧返回应该用于"返回上一层"的场景
- 单独左侧返回会让用户不知道如何关闭弹窗

正确做法：左侧返回必须和右侧关闭同时使用
