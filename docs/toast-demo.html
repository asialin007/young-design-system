Toast 轻提示

Toast 用于显示短暂的通知消息，在屏幕中央或特定位置显示，自动消失。 适用于操作反馈、状态提示等场景。

默认 Toast

单行文字

简单的单行提示信息

操作成功

多行文字

支持两行以内的文字显示

建议内容
不超过两行
<!-- HTML -->
<div class="yds-toast yds-toast--default">
  <span class="yds-toast__text">操作成功</span>
</div>

品牌 Toast

带头像

显示品牌头像和内容

只显示标题，不显示副标题

VIPSHOP

    <img src="avatar.jpg" alt="Avatar">
  </div>
  <div class="yds-toast__content">

JavaScript 使用示例

Toast 工具函数

封装 Toast 显示工具函数

// Toast 工具函数
function showToast(text, variant = 'default') {
  const toast = document.createElement('div');
  toast.classList.add('yds-toast', `yds-toast--${variant}`, 'yds-toast--fade-in');
  
  let html = '';
  switch (variant) {
    case 'brand':
      html = `<div class="yds-toast__avatar">
        <img src="avatar.jpg" alt="Avatar">
      </div>
      <div class="yds-toast__content">
        <span class="yds-toast__title">VIPSHOP</span>
        <span class="yds-toast__subtitle">${text}</span>
      </div>`;
      break;
    case 'atmosphere':
      html = `<img src="../icons/y_toast_decoration_left.svg" class="yds-toast__icon yds-toast__icon--left">
      <span class="yds-toast__text">${text}</span>
      <img src="../icons/y_toast_decoration_right.svg" class="yds-toast__icon yds-toast__icon--right">`;
      break;
    default:
      html = `<span class="yds-toast__text">${text}</span>`;
  }
  
  toast.innerHTML = html;
  document.body.appendChild(toast);
  
  // 2秒后消失
  setTimeout(() => {
    toast.classList.remove('yds-toast--fade-in');
    toast.classList.add('yds-toast--fade-out');
    toast.addEventListener('animationend', () => {
      toast.remove();
    });
  }, 2000);
}
