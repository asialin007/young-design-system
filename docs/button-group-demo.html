Button Group 按钮组合

Button Group 按钮组合组件用于组合多个按钮，支持多种对齐方式和尺寸配置。 适用于表单操作、工具栏、页面导航等场景。

按钮默认色

所有主按钮、次按钮和更多按钮均使用默认的黑色样式

单个按钮

单按钮

count=1，只显示一个按钮，使用黑色

两个按钮 - 水平对齐

水平对齐

count=2，主按钮和次按钮宽度相等，使用黑色

两个按钮 - 右侧对齐

右侧对齐

count=2，主按钮占满宽度，次按钮靠右显示，使用黑色

两个按钮 - 文字混合对齐

文字混合

count=2，主按钮和次按钮都带有图标，使用黑色

三个按钮

三按钮

count=3，主按钮 + 次按钮 + 更多按钮，均使用黑色

四个按钮

四按钮

count=4，主按钮 + 次按钮 + 三级按钮 + 四级按钮，均使用黑色

垂直排列
